## Mini To-Do List App

This is a beginner Python project.

## What it does
- Shows tasks
- Adds tasks
- Removes tasks
- Saves tasks in a file

## How to run it

1. Open the project folder
2. Run this command:

python main.py

## Files
- main.py = the code
- tasks.txt = saved tasks
- README.md = project instructions
